# Palettes

Named, consistent material sets used across a build to enforce a coherent
look. A palette specifies:

- **Primary stone**: the bulk of the walls
- **Trim stone**: string-courses, corners, window surrounds
- **Roof material**: stairs/slabs for pitched roofs
- **Wood**: framing, doors, lintels, furniture
- **Glass**: panes (plain, white-tinted, colored, etc.)
- **Accent**: banners, candles, foliage — the punctuation

A generator should pick ONE palette for a build and stick to it.
