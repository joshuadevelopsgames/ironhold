# King's Console — Art Direction Kit

Everything an artist (or AI image model + cleanup pass) needs to paint the
console GUI. The realm map is intentionally **out of scope** — leave that
slot blank in the atlas; the screen draws it from code.

## Pipeline

1. Read [`style-rules.md`](style-rules.md) — non-negotiable.
2. Load [`palette.gpl`](palette.gpl) into Aseprite as Indexed mode. Every
   sprite must use only these 16 colors.
3. Open `atlas-template.png` (generated by `generate-template.py`). Each
   labeled rectangle is a sprite slot — paint inside, do not change size.
4. For each sprite, read the matching prompt in [`prompts/`](prompts/).
   The prompt names the constraints; if a generated image violates them,
   reject and regenerate.
5. Save the finished atlas as `kings_console.png` and drop it into
   `src/main/resources/assets/ironhold/textures/gui/console/`.
6. Flip `KingsConsoleScreen.USE_ATLAS = true` and reload — the screen
   has `blitSlot(...)` and `blitSlotStretch(...)` helpers wired to every
   UV in `ConsoleAtlas.java`.

## Out of scope

These mocked-up sections were **removed from the screen** and need no
art:

- **Realm map** — the painter doesn't paint it. The screen has no map
  area anymore.
- **Unrest meter** — the screen has no unrest section anymore. Atlas
  slot for the skull (`(112, 0)`) is reserved/blank.

## What's in this kit

| File | Purpose |
|------|---------|
| `palette.md` / `palette.gpl` | Locked 16-color palette. |
| `style-rules.md` | Outline, shading, scale, dither rules. |
| `state-requirements.md` | Which sprites need idle/hover/pressed/disabled. |
| `atlas-spec.md` | Every sprite, its UV box, its size, what it's for. |
| `generate-template.py` | Renders `atlas-template.png` from `atlas-spec.md`. |
| `atlas-template.png` | Visual layout sheet — paint into the labeled slots. |
| `prompts/` | One MD file per sprite group with a ready-to-paste prompt. |

## Atlas size

Single PNG, **256 × 256**, transparent background (RGBA).
All sprites packed in the top half (y ≤ 136); bottom half reserved for
future expansion.
